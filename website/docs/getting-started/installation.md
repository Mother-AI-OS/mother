# Installation

Get Mother AI OS up and running in minutes.

## Requirements

- Python 3.11 or higher
- An Anthropic API key ([get one here](https://console.anthropic.com/))

## Install from PyPI

The easiest way to install Mother AI OS:

```bash
pip install mother-ai-os
```

## Install from Source

For the latest development version:

```bash
git clone https://github.com/Mother-AI-OS/mother.git
cd mother
pip install -e .
```

## Verify Installation

Check that Mother AI OS is installed correctly:

```bash
mother --version
# Mother AI OS v0.1.0
```

## Next Steps

Now that you have Mother AI OS installed, [configure your environment](/docs/getting-started/configuration) to start using it.
